# Komandkalkulator

## To install:
- run "npm install" in the "projekt04" folder
- run "npm run create_env". neccessary.
- "npm run setup_database" to create the database.
- to fill with dummy data run "npm run fill_db"
- to run the server "npm run run_server"
To create an admin account, run "npm run create_admin_account [username] [password]". It is required to edit user permissions and make them able to do things.

Port is 1965 by default. Change the env file if you want it otherwise. If no port is specified, it runs on 6767.


Editing, adding requires existing entries because it just does.
